# BetterEnd-Eternal Fabric 1.21.1

A complete port of BetterEnd to Minecraft 1.21.1, adding new biomes, blocks, items, and mobs to the End dimension.

## Installation

### Requirements
- **Minecraft** 1.21.1
- **Fabric Loader** (latest)
- **Fabric API** (latest for 1.21.1)
- **BCLib** 21.0.13
- **Wover** 21.0.13

### Steps
1. Install Fabric Loader for 1.21.1
2. Download and install Fabric API, BCLib, and Wover
3. Download BetterEnd from [Modrinth](link) or [CurseForge](link)
4. Place the `.jar` file in your `mods/` folder
5. Launch Minecraft!

## Bug Reports

Found a bug? Please report it on our [Issue Tracker](https://github.com/Darkcloak737/BetterEnd-Eternal/issues)!

**Before reporting:**
- Check if it's already been reported
- Include your Minecraft version, mod version, and crash log
- List any other mods you have installed

## Building from Source

```bash
git clone https://github.com/YourUsername/BetterEnd.git
cd BetterEnd
./gradlew build
```

The compiled `.jar` will be in `build/libs/`
